# freight-data-cleaner

Beginner Python project for cleaning and processing Excel/CSV datasets.  
Focus on data validation, basic transformations, and generating clean outputs for analysis.

## What it does
- Loads a raw CSV dataset
- Standardizes column names
- Removes rows with missing values
- Converts the `amount` column to numeric (invalid values become NaN)
- Exports a cleaned CSV file
